# Учёт подписок (в браузере)

Это небольшой учебный проект: приложение для учёта текущих подписок на месяц.

## Что есть в проекте

- `index.html` — приветственный лендинг (главная страница)
- `Приложение подписок.html` — рабочее приложение: добавление/удаление подписок, подсчёт общей суммы и очистка всех данных

## Как запустить

1. Откройте `index.html` в браузере.
2. Нажмите кнопку «К подпискам», чтобы перейти к функционалу.

> Данные сохраняются в `localStorage`, поэтому регистрация/сервер не нужны.

## Тема (светлая/тёмная)

Кнопка переключения темы находится в правом нижнем углу. Выбор сохраняется в `localStorage`, поэтому действует и на лендинге, и на странице приложения.

## Замечания

Проект сделан как статические HTML-страницы без сборщиков и зависимостей.
